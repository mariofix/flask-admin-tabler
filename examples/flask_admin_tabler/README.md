# Flask-Admin Tabler Theme Example

This example demonstrates a custom Tabler UI theme for Flask-Admin using SQLAlchemy models.

## How to run this example

Clone the repository and navigate to this example:

```shell
git clone https://github.com/mariofix/flask-admin-tabler.git
cd flask-admin-tabler/examples/flask_admin_tabler
```

> This example uses [`uv`](https://docs.astral.sh/uv/) to manage its dependencies and developer environment.

Run the example using `uv`, which will manage the environment and dependencies automatically:

```shell
uv run main.py
```
